# Chat-App-Real-Time

Front-End: ReactJs, Chakra-ui, LottieFiles, Socket.io-client.

Back-End: NodeJs, JWT, Socket.io, MongoDB, NodeMailer.

Features:

	Sign In, Sign Up (upload avatar), Log Out.
	
	View Profile.
	
	Chat 1-1 (user - user).
	
	Chat 1-n (group chat).
	
	Typing Messages.
	
	Update, Delete, Add User In a Group (only admin).
	
	Update, Delete, Add a Group Chat.
	
	Leave a Group Chat.
	
	Rename a Group Chat.
	
	Search Members.
	
	New Message Notifications.
	
	Sign In To Your Email To Verify Your Account.
